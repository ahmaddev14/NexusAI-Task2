# To-Do List App

<img width="1789" height="814" alt="Screenshot_46" src="https://github.com/user-attachments/assets/1142ba84-860f-43e6-bfba-77086808cac8" />

Simple task management app with localStorage

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark as complete
- Clear completed
- Data persists

## Structure

```
todo-app/
├── index.html
├── css/style.css
└── js/app.js
```

## Usage

1. Open index.html
2. Type task and click Add
3. Check box to complete
4. Edit or Delete as needed
5. Clear Done removes completed

## Tech

- HTML5
- CSS3
- JavaScript
- localStorage

## Functions

- add() - adds new task
- toggle() - marks done/undone
- edit() - changes task text
- remove() - deletes task
- clearDone() - removes completed
- save() - saves to localStorage
- render() - displays tasks
